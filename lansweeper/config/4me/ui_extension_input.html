Go to your Lansweeper profile and select the 'Developer Tools' menu option.
Select the 'All applications' menu and add a new application with:
  • Application name: 4me
  • Allowed callback URL: https://example.com
  • Developed by: 4me
Press 'Add Application' and open the '4me-credentials.txt' file that Lansweeper generated.
Copy the 'Client ID' and 'Client Secret' values from the file to the fields below.
Then press 'Save' below and await further instructions.
If no instructions are provided after 5 seconds, please refresh your browser.
Copy the value below to Lansweeper in the 'Allowed callback URL' field of the newly created 4me application.
Then press the 'Save' button at the top right.
Press the 'Authorize' button and select all Lansweeper Sites that need to be synced to 4me.
Finally press 'Allow' after which 4me will establish the connection to Lansweeper.
